# Description
GrubCut: Markov Random Fileds in Image Segmentation

# Usage

```
git clone https://github.com/shota-takayama/grabcut_py.git
cd grabcut
python main.py
```

# Requirements

* numpy 1.11.2
* scikit-learn 0.18
* graph-tool 2.19
* opencv 2.4.12
